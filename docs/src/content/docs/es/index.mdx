## Qué hace

<CardGrid stagger>
    <Card title="Implementar Integridad de Subrecursos" icon="pencil">
        Astro-Shield se encarga de calcular los hashes SRI y de establecer el
        atributo `integrity` en las etiquetas de script y estilo por ti.
    </Card>
    <Card title="Establecer encabezados de Política de Seguridad de Contenidos" icon="seti:lock">
        Astro-Shield puede establecer automáticamente los encabezados de `Content-Security-Policy`
        por ti.
    </Card>
    <Card title="Bloquear scripts inyectados" icon="close">
        Cuando Astro-Shield detecta un script sospechoso (es decir, probablemente inyectado por
        un atacante), lo eliminará del HTML renderizado.
    </Card>
