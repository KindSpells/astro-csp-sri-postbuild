## Что делает Astro-Shield?

<CardGrid stagger>
	<Card title="Реализует Subresource Integrity" icon="pencil">
		Astro-Shield берет на себя расчет хэшей SRI и установку атрибута `integrity` для тегов script и style за вас.
	</Card>
	<Card title="Устанавливает Content-Security-Policy заголовки" icon="seti:lock">
		Astro-Shield может автоматически выставлять заголовки `Content-Security-Policy` за вас.
	</Card>
	<Card title="Блокирует внедрённые вредоносные скрипты" icon="close">
		Когда Astro-Shield обнаруживает вредоносный скрипт (который, например, мог быть вставлен злоумышленником), то он удаляет его из отрендеренного HTML.
	</Card>
